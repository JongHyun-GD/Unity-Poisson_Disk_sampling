# Unity-Poisson_Disk_sampling
## 목적
- Unity3D에서 Simplified  Poission Disk Sampling(PDS) 알고리즘을 통해 자연스러운 Object Placement를 구현한다.

## PDS 활용
- PDS 알고리즘은 3D environment 자동 조성, image 생성 등 다양한 분야에 쓰일 수 있다.
